# Secret Santa Name Grouper

A simple web application to randomly assign Secret Santa gift recipients.

## How to Use

1. Enter the names of participants, one name per line in the text area.
2. Click "Generate Assignments" to create random pairings.
3. Each person will be assigned to give a gift to another person.

## Deployment

This project is set up to deploy automatically to GitHub Pages using GitHub Actions.

### To deploy:

1. Create a new repository on GitHub.
2. Push this code to the `main` branch of your repository.
3. Go to your repository settings.
4. Under "Pages", set the source to "GitHub Actions".
5. The workflow will automatically deploy your site.

Your site will be available at `https://<username>.github.io/<repository-name>/`

## Local Development

Open `index.html` in your web browser to test locally.
